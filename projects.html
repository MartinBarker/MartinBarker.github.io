www.discogstagger.site
Github

A ruby on rails website which uses the discogs-wrapper api gem to generate a list of metadata tags for a discogs release URL. Usefull for YouTube music uploaders to use instead of manually typing out tags for a discogs release.

www.tagger.site
Github

A website which generates track-list time-stamps for music files. Albums will be arranged in a format which lists the timestamps for each song. The taget use for this program is to help full album uploads on YouTube. Previously this tracklisting would have to be computed tediously by hand.

VLC-Metadata

A python script which updates song metadata and album art from VLC media player for streaming software like OBS to display. All of the text in this image is automatically updated whenever a new song plays in VLC media player, metadata such as song name, album, artist, genre and year are displayed.
