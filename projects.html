VLC-Metadata

A python script which updates song metadata and album art from VLC media player for streaming software like OBS to display. All of the text in this image is automatically updated whenever a new song plays in VLC media player, metadata such as song name, album, artist, genre and year are displayed. The album artwork is also resized and saved in a new folder. An example of this script in use can be seen here: https://www.youtube.com/watch?v=eTXGcsqmj3g

Tracklist Timestamp Generator

A website which generates track-list time-stamps for music files. Albums will be arranged in a format which lists the timestamps for each song. The taget use for this program is to help full album uploads on YouTube. Previously this tracklisting would have to be computed tediously by hand.

VLC-Playlist-Randomizer
